# PDF 工具箱

三合一 PDF 工具，部署在 Streamlit Community Cloud：

| 分頁 | 功能 |
|------|------|
| 📝 轉 Markdown | 將 PDF 轉為 Markdown |
| ✂️ 拆分頁面 | 輸入頁碼範圍，抽取指定頁面輸出新 PDF |
| 🔗 合併 PDF | 上傳多個 PDF，依序合併成一個檔案 |
| 🖊️ 編輯頁面 | 預覽縮圖，上下移動重排或刪除頁面 |

## 本地執行

```bash
pip install -r requirements_cloud.txt
streamlit run app_cloud.py
```
